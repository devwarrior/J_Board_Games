# JavaHanoi

JavaHanoi is not a game, it's only a demo of the Hanoi Tower. The program tries to solve the Hanoi Tower with the number of pieces the user specifies. It provides an example about how it is possible to handle animations in Java.

# How to compile
Enter the src directory and compile the source code using the following command:

`javac Hanoi.java`

nothing more

# How to run
From the source directory issue the following command:

`java -classpath . Hanoi <number_of_pieces> <animation_speed>`
